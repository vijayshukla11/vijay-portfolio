# **App Name**: Fridge Chef

## Core Features:

- Ingredient Input: Allow users to input ingredients they have available.
- Recipe Generation: Generate recipe suggestions based on the user's available ingredients, using a tool to analyze ingredients and select relevant recipes.
- Recipe Display: Display generated recipes in a clear, easy-to-read format.
- Additional Information: Show instructions, nutritional information, and cooking time for each recipe.

## Style Guidelines:

- Primary color: Fresh green (#8FBC8F) to align with the theme and evoke a sense of freshness.
- Background color: Light pale green (#F0FFF0), offering a soft, clean backdrop.
- Accent color: Earthy olive (#556B2F), adding contrast for key actions and elements.
- Body and headline font: 'PT Sans', a humanist sans-serif that is modern with a touch of warmth, works well for both headlines and body text.
- Use simple, outlined icons for ingredients and recipe categories.
- Keep the layout clean and intuitive with clear sections for inputting ingredients and viewing recipes.
